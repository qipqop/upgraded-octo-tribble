# Инструкция по запуску бота на сервере

1. **Подготовка сервера**
   Убедись, что сервер настроен и имеет установленный Python версии 3.8 или выше. Проверить версию можно командой:
   ```bash

   python3 --version

   ```

2. **Клонирование или загрузка проекта**

   Если проект передается архивом, распакуй его:
   ```bash

   unzip fastMarket.zip
   cd fastMarket

   ```

3. **Установка зависимостей**
   Убедись, что в папке проекта есть файл `requirements.txt` (или `reqs.txt`). Установи зависимости:
   ```bash

   pip install -r reqs.txt

   ```

4. **Запуск бота**
   Найди главный файл бота, обычно это `bot.py` или аналогичный. Запусти его:
   ```bash

   python bot.py

   ```

6. **Проверка работы**

   Если бот запустился успешно, в консоли появится информация, подтверждающая это. Проверь, работает ли бот в Telegram.

## Основные команды Linux ##
- **Перейти на уровень выше в каталоге**
   ```bash
   cd ../
   ```

- **Обновление пакетов**
   ```bash
   sudo apt update && sudo apt upgrade
   ```

- **Установка Python**
   Если Python не установлен:
   ```bash
   sudo apt install python3 python3-pip
   ```

- **Установка дополнительных библиотек**
   Если требуется, можно устанавливать библиотеки отдельно:
   ```bash
   pip install <имя библиотеки>
   ```

- **Проверка процесса**
   Узнать, работает ли бот:
   ```bash
   ps aux | grep python
   ```

- **Остановка бота**
   Найди PID процесса бота и убей его:
   ```bash
   kill <PID>
   ```

- **Запуск бота в фоновом режиме**
   Чтобы бот продолжал работать после закрытия терминала:
   ```bash
   nohup python3 bot.py &
   ```

---

## Примечания:
1. Главный файл проекта — это `bot.py`.
2. Если возникают проблемы, проверь ошибки в консоли. Например, если библиотека не установлена, попробуй переустановить зависимости через `pip install`.
